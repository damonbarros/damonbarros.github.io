    Sobre Mim

    Desenvolvedor WEB

    Meu nome completo é Francisco Damon Victor Barros Leão, mais conhecido como Damon Barros, tenho 17 anos, sou de Croatá, distrito da cidade de São Gonçalo do Amarante e atualmente estou cursando o 3° ano do Ensino Médio. Como desenvolvedor web, meu foco é criar soluções digitais incríveis e funcionais. Estou constantemente atualizado sobre as mais recentes tendências e tecnologias da web, buscando aprimorar minhas habilidades para entregar projetos de alta qualidade. Com atenção aos detalhes e uma mentalidade orientada para o usuário, estou sempre em busca de desafios estimulantes e oportunidades de aprendizado. Espero estar sempre ajudando a sociedade a combater problemas sociais com aplicações inovadoras, e garanto que estarei sempre focado nesse objetivo de contribuir de uma forma positiva para o mundo através da programação.

    Linguagens utilizadas por mim no dia a dia:


    Capacidade de criar estrutura e semântica para páginas web, utilizando as principais tags e elementos do HTML, como cabeçalhos, parágrafos, listas, imagens, links, entre outros.

    Habilidade para estilizar e dar aparência visual às páginas web, utilizando seletores, propriedades CSS e conhecendo conceitos como layout, cores, tipografia, estilização de botões e menus, responsividade, entre outros.

    Conhecimento básico de JavaScript para adicionar interatividade às páginas web, manipular elementos do DOM, lidar com eventos, criar animações simples, realizar validações de formulários, entre outros.

    Compreensão básica de PHP, uma linguagem de programação do lado do servidor, para construir recursos dinâmicos em um site ou aplicação web, como formulários de envio, processamento de dados, conexão com bancos de dados e geração de conteúdo dinâmico.

    Experiência

    Ao longo da minha jornada como estudante do curso técnico da escola Walter Ramos de Araújo, fiz muitas atividades intuitivas que me ajudaram a desenvolver experiência na área de programação. Estou constantemente procurando adquirir ainda mais conhecimentos necessários para me tornar um programador mais efetivo, que possua habilidades para resolver todos os tipos de problemática. O projeto Redatrix (projeto de conclusão do curso), disponível no meu perfil do GitHub (damonbarros) foi o site mais completo feito por mim durante a tragetória estudantil ao longo desses 3 anos, possuindo diversas funcionalidades, aonde tenho certeza que a aplicação ajudará vários alunos. Essse trabalho de conclusão do curso técnico foi bastante prazeroso, pois me permitiu adquirir experiências como entrevista com cliente, modelagem do banco de dados, entre outras habilidades que foram aprimoradas ao longo deste processo.
